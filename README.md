# Sitio web RECA

## Instrucciones para desplegar local

Instalar ruby (recomendado por medio de rvm o rbenv). Luego ejecutar:

```
gem install jekyll bundler
bundle install
bundle update --bundler
bundle exec jekyll serve
```
